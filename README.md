## Running Tests

Use these commands from the repository root:

1. Install dependencies

	pip3 install -r requirements.txt

2. Run all tests

	pytest -v

3. Run backend API tests only

	pytest -v tests

4. Re-run backend tests to check stability

	pytest -v tests
